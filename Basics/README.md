# 🧠 Python Data Structures

This folder contains programs based on core Python data structures: **Strings, Lists, Tuples, Sets, and Dictionaries**.

---

## 📚 Concepts Overview

* **Strings** 🔤
  Used to store text data (immutable)

* **Lists** 📋
  Ordered collection, **mutable** (can change)

* **Tuples** 📦
  Ordered collection, **immutable** (fixed)

* **Sets** 🔗
  Unordered collection, **no duplicate values**

* **Dictionaries** 📖
  Store data in **key-value pairs**

---

## 🎯 Purpose

* Understand different ways to store data
* Learn when to use each data structure
* Build strong programming fundamentals
